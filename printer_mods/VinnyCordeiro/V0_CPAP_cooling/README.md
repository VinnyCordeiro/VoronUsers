# CPAP cooling for the V0

Took the inspiration from another user on the Jubilee Discord server.

The V0 is known to not print PLA well because of the standard part cooling fans, which are tiny (like the rest of the printer). Because of size restrictions bigger fans can't be added without sacrificing print volume, which is already small, so using CPAP tubes connected to a more powerful cooling solution outside of the toolhead or even the gantry is ideal: CPAP tubes are very light and should not add significant mass to the printing gantry. My idea is to connect both sides cooling ducts with [this adapter](https://www.thingiverse.com/thing:3679095).
