# V0 Z motor mount for 10mm flat-to-flat M5 nuts

Standard M5 nuts have, by the ANSI/ASME B18.2.4.1M and B18.2.4.2M standards, a flat-to-flat size between 7.78mm and 8.0mm. However, it isn't uncommon to find a M5 nut with a flat-to-flat size of up to 10mm. This mod is for those who already have these nuts and didn't bought the standard ones, hoping to save time and money.

If you are going to build a Voron 0 from scratch and still need to buy all the fasteners, pay attention and buy the standard M5 nut, so that this mod is unnecessary.
