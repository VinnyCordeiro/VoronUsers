# V0 XY joints using 20 teeth GT2 idlers

Credits for Voron user **shay** for the original idea.

It is *very* easy to overtighten the belts on a V0, and the spec 16 teeth idlers' ball bearings simply can't take that load, turning into dust in a few hours. Gates, the creator of the GT2 (or would it be 2GT?), does **not** manufactures 16 teeth idlers, just 20 teeth up, so it is easier to find 20 teeth idlers made with quality ball bearings.

This mod makes it possible to use 20 teeth idlers on a V0, by modifing the XY joints on both left and right sides. I took care and, at least on CAD, the belt's path is unchanged, meaning that no other modifications are required on the V0.

## Updates:
April 07, 2021: Initial upload. Included are both STL and STEP files of modified XY joints, the mod was made on Fusion 360.
